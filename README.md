FinalProject2-Secure-Gallery.

Назначение: Веб-приложение для безопасного хранения и управления личными изображениями с регистрацией и авторизацией пользователей.

Проект представляет собой защищенное веб-приложение-галерею, разработанное на основе Spring Boot. Оно позволяет пользователям регистрироваться, входить в систему и загружать/просматривать свои изображения и других пользователей. Основной фокус сделан на использовании Spring Security и разделении доступа к файлам.

Предназначение: Для частных пользователей, которым требуется простой и безопасный способ хранения личных фотографий в облаке с контролем доступа.

Основные функции (MVP):

Регистрация и Авторизация пользователей (Spring Security).
Загрузка изображений.
Хранение метаданных изображений и пользовательских данных в базе данных.
Просмотр загруженных изображений.
Обработка ошибок при аутентификации/авторизации.
