# План тестирования проекта FinalProject2-Secure-Gallery

## 1. Цели тестирования
1.  Функциональная валидация всех требований (Регистрация, Вход, CRUD изображений).
2.  Проверка надежности и корректности конфигурации Spring Security.
3.  Оценка удобства пользовательского интерфейса (Usability).

## 2. Область покрытия
* **Модули:** Аутентификация, Управление пользователями, Загрузка и Хранение файлов, Отображение галереи.
* **Типы:** Функциональное тестирование (FT), Пользовательское тестирование (UT), Тестирование безопасности (Security).

## 3. Среда и Инструменты
* **Среда:** Локальный Spring Boot Server, H2/PostgreSQL, Windows/Linux.
* **Инструменты:** Браузеры (Chrome), Postman/cURL, Unit-тесты (JUnit 5).

## 4. Критерии завершения
* Успешное выполнение не менее **85%** Функциональных тест-кейсов.
* Отсутствие **критических** ошибок (Critical severity) при тестировании безопасности.
